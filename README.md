# Automatic Vehicle License Plate Recognition and Access Control System Using Deep Learning.
This project aims to build a critical enabling technology for intelligent transportation systems, campus access control, and smart parking infrastructure. YOLO11n offers the best overall balance and was selected for system integration. Used EasyOCR for end to end character recognition. Before that, some preprocessing(e.g. deblurring, transformation) has been done. Finally, deployed within a full-stack web application. The system features three distinct roles for users (Admin, Staff, User), a vehicle registration and approval workflow, real time recognition event streaming via Django Channels and Redis WebSockets, and three camera input modes: image file upload, live webcam, and IP camera stream.
